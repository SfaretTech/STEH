# **App Name**: STEH Edu

## Core Features:

- Course Catalog: Display a list of available courses, filtering by categories such as 'Web Technologies', 'Graphics Design', 'IoT', and 'Microsoft Packages'.
- Course Details: Show course details, including description, fees, duration, and grade level.
- Inquiry Form: Provide a contact form or button for inquiries about courses.
- About Section: Display the mission statement, partnership goals, and other information about SFARET TECH EDU HUB.
- AI Course Advisor: AI-powered course recommendation tool: based on user-submitted interests, recommend appropriate courses at STEH.

## Style Guidelines:

- Primary color: Vibrant purple (#A050BE) to reflect the Sfaret Technologies brand, transitioning from the original logo's background.
- Background color: Light purple (#F1E5F7), providing a soft and clean backdrop that complements the primary color.
- Accent color: Blue (#50A0BE) to highlight calls to action and important information, providing a clear visual contrast and signaling interactivity.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-forward, modern aesthetic, suitable for titles and emphasis. Body font: 'Inter' (sans-serif) to ensure readability and a clean user experience.
- Use a consistent set of vector icons throughout the website to represent course categories, features, and important information.
- Design a responsive layout that adapts seamlessly to different screen sizes, ensuring a consistent experience across devices.
